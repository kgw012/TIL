# 7/15(목) 정리



## git 명령어 모음

``` ba
git init
git add .
git commit -m 'commit message'

git remote add <repo 이름 지정(보통 origin)> <repo url>
git branch -M main      	(브랜치 이름 master에서 main으로 변경)
git push -u origin main 	(main 브랜치를 origin이라는 이름의 remote repo로 push)
    		-> (-u 옵션을 사용하면, 후에 push할때 'git push'만 쳐도 자동으로 push 됨)
    
git clone <repo url>
```
